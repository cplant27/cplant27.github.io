Welcome to TOAL

TOAL is a modern programming language designed for simplicity and ease of use. It is perfect for beginners and experienced programmers alike.

With TOAL, you can create powerful programs and applications using a simple and intuitive syntax. Whether you're building a small utility or a large-scale system, TOAL has everything you need to get the job done.

So why wait? Start coding with TOAL today and discover the joy of programming!

Features

  • Simple and intuitive syntax
  • High-level constructs for easy programming
  • Object-oriented design for modularity and flexibility
  • Fast and efficient execution
  • Compatible with popular libraries and frameworks

Syntax

TOAL uses a simple and intuitive syntax that makes programming a breeze. Here's an example:

          
            automate greet(name) {
              print "Hello" + name + "!";
            }

            greet("World");
          
        
          
            make x with 1;
            
            add 1 to x;
            subtract 1 from x;
            multiply x by 1;
            divide x by 1;
            
            change x to 5;
            print x plus 5;
