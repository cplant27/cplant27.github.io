Welcome to T.O.A.L.

The Totally Original and Applicable Language (T.O.A.L.) is a modern programming language designed for simplicity and ease of use. It is perfect for beginners and experienced programmers alike.

With T.O.A.L., you can create powerful programs and applications using a simple and intuitive syntax. Whether you're building a small utility or a large-scale system, T.O.A.L. has everything you need to get the job done.

So why wait? Start coding with T.O.A.L. today and discover the joy of programming!

Features

  • Feel like a pro with our simple syntax and commands
  • Contain all the data you need with our built in list structure
  • Easily and elegantly repeat tasks or iterate over data with loops
  • Make decisions with ease using conditional statements
  • Automate your code and call it anytime you want with custom automations

Syntax

T.O.A.L. uses a simple and intuitive syntax that makes programming a breeze. Here are some examples:

          
            # Automations make coding easy!

            automate greeting(name) {
              print "Hello " + name + " !";
            }
            greeting("World");
          
        
          
            # Simple variable declaration

            make x with 1;
            make y with "Make a variable with ANY type!"
            make z with ["even", "a", "list"]

            # and even simpler incrementation
            
            add 1 to x;
            subtract 1 from x;
            multiply x by 1;
            divide x by 1;

            # Say goodbye to those confusing math symbols!
            
            change x to 5;
            make a with 2 plus x minus 7;
            make b with 2 to the 9 minus (y divided by x) times x to the 0;
          
        
          
            # While loops
            
            make x with 0;
            loop while x is less than 5 {
                add 1 to x;
                break;
            }
            print x;

            # For loops
            
            make this_list with [1, "hello", [1,2,3], []];
            loop over element in this_list {
                print element;
            }
          
        
          
            # Even conditionals

            make x with 5;
            if x is 5 { 
                print "x is 5";
            }
            if not {
                print "x is not 5";
            }
